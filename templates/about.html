🤖 About AI Career Matchmaker

Intelligent Career Recommendation Platform

📌 Project Overview

AI Career Matchmaker is an intelligent career recommendation platform that helps users discover the most suitable technology careers based on their skills.


Using Natural Language Processing (NLP), TF-IDF Vectorization and Cosine Similarity, the system compares user skills with industry career profiles and recommends the best matching career paths.


🚀 Features

  • AI Based Career Recommendation
  • Skill Match Analysis
  • Matched & Missing Skills Detection
  • Career Roadmap Generator
  • Salary Insights
  • Demand Analysis
  • Growth Prediction
  • Career Overview
  • PDF Career Report Download
  • Interactive Glassmorphism UI

🛠 Technologies Used

  • Python
  • Flask
  • Pandas
  • Scikit-Learn
  • TF-IDF Vectorizer
  • Cosine Similarity
  • HTML5
  • CSS3
  • JavaScript
  • ReportLab PDF Generator
